# AI-Powered Tour Planner - Diagrams

This directory contains visual representations of the system architecture, data flow, and key processes in the AI-Powered Tour Planner.

## Diagram List

1. **System Architecture** (`system_architecture.puml`)
   - High-level overview of the system components
   - Shows interaction between frontend, backend, and external services

2. **Data Flow** (`data_flow.puml`)
   - Illustrates how data moves through the system
   - Shows the sequence of operations from user input to itinerary generation

3. **AI Recommendation Engine** (`recommendation_engine.puml`)
   - Details the AI/ML components and their interactions
   - Shows the flow of data through the recommendation pipeline

4. **User Journey** (`user_journey.puml`)
   - Maps the typical user flow through the application
   - Shows key interactions and decision points

5. **Deployment Architecture** (`deployment.puml`)
   - Shows the production deployment setup
   - Includes load balancing, caching, and database replication

## How to Use

1. Install [PlantUML](https://plantuml.com/) or use an online editor
2. Open the `.puml` files to view or edit the diagrams
3. Export to PNG/SVG using your preferred PlantUML tool

## Tools

- [PlantUML](https://plantuml.com/) - For creating and editing diagrams
- [PlantText](https://www.planttext.com/) - Online PlantUML editor
- [VS Code PlantUML Extension](https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml) - For editing in VS Code
